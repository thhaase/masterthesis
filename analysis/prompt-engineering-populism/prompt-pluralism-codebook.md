# Codes
**4 scoring dimensions**, each on a -3 to +3 scale:
- bipolar scale (-3 to +3) for all four dimensions 
	- pro-pluralist (+)
	- anti-pluralist (-)

| Dimension                              | What it captures                                       | Key traps to prevent false positives             |
| -------------------------------------- | ------------------------------------------------------ | ------------------------------------------------ |
| **Respect for Opponents**              | Demonization vs. civility                              | Disagreement Trap, Tone Trap, Hyperbole Trap     |
| **Commitment to Democratic Processes** | Attitudes toward elections, free speech/press/assembly | Reform Trap, Criticism Trap, Authority Trap      |
| **Respect for Minority Rights**        | Majority override vs. rights protection                | Majority Trap, Policy Trap, Identity Trap        |
| **Stance on Political Violence**       | Encouragement vs. condemnation                         | Metaphor Trap, Reporting Trap, Self-Defense Trap |
